<h3>课程介绍：</h3>
<p>自动驾驶是当前人工智能领域最受关注的热点行业，让众多技术工作者梦想加入这个充满想象力的赛道。然而，自动驾驶是一门复杂的交叉学科，涉及领域种类众多且技术栈跨度大，对于初学者而言如果没有行业导师指导，很难自学。本课程从浅入深讲解技术与算法，带你系统入门自动驾驶四大模块，感知、规划、决策和控制。</p>
